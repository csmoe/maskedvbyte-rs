maskedvbyte-sys
=========
Bindings for [MaskedVByte](https://github.com/lemire/MaskedVByte).

### Documentation

https://docs.rs/maskedvbyte-sys


### Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
maskedvbyte-sys = "0.1"
```

then add this to your crate root:

```rust
extern crate maskedvbyte_sys;
```
